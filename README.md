# Домашнє завдання. Основи аналізу та алгоритмічні стратегії

Модуль присвячений підходу «розділяй і володарюй» та жадібним алгоритмам.

---

## Завдання 1. Пошук максимального та мінімального елементів

**Файл:** `task1.py`

Функція `find_min_max` знаходить мінімальний і максимальний елементи масиву методом
**«розділяй і володарюй»**:

- масив рекурсивно розбивається навпіл, доки не залишиться один або два елементи (базові випадки);
- результати підмасивів об'єднуються порівнянням їхніх мінімумів та максимумів;
- рекурентне співвідношення `T(n) = 2·T(n/2) + O(1)` дає лінійну складність **O(n)**.

Функція приймає масив довільної довжини та повертає кортеж `(мінімум, максимум)`.

## Завдання 2. Оптимізація черги 3D-принтера

**Файл:** `task2.py`

Функція `optimize_printing` формує чергу 3D-друку **жадібним алгоритмом** з урахуванням
пріоритетів та технічних обмежень принтера.

Пріоритети завдань:
1. Курсові/дипломні роботи (найвищий);
2. Лабораторні роботи;
3. Особисті проєкти (найнижчий).

Алгоритм:
1. вхідні дані перетворюються на об'єкти `dataclass` — `PrintJob` та `PrinterConstraints`;
2. завдання стабільно сортуються за пріоритетом (порядок у межах одного пріоритету зберігається);
3. моделі об'єднуються в групи для одночасного друку, доки не перевищено `max_volume` та `max_items`;
4. час групи дорівнює максимальному часу серед моделей у ній.

Функція повертає словник із порядком друку (`print_order`) та загальним часом (`total_time`).
